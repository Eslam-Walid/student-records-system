It's a student records system based on C programming languages.
The system provides two modes, user mode and admin mode.
The admin can modify and access any data in the system by entering his password to the system to get all these privileges.
The user can only access his info and view them by entering his ID and password.
--------------------------------------------------------------------------------------------------------------------------
The default admin password to the system is "0000".
--------------------------------------------------------------------------------------------------------------------------
The first version is based on array of pointers to struct.
The second will have database to store data before terminating the program and will be based on linked list.
